# Superhive Submission Checklist

- [ ] Extension ZIP passes Cloth NeXt release policy and artifact scan.
- [ ] ZIP contains no PPF solver binary, archive, DLL or runtime.
- [ ] GPL-3.0-or-later is selected and source is provided to customers.
- [ ] Quick Start `.blend` opens in the current official Blender release.
- [ ] Demo objects, materials, collections and scenes use descriptive English names.
- [ ] Demo has no missing external assets and renders without relinking.
- [ ] Written installation, usage, requirements, limitations and FAQ are pasted on-site.
- [ ] At least five original PNG/JPG images are 2:1 and at least 1200 px wide.
- [ ] Product imagery does not use Blender branding as a design element.
- [ ] Supported Blender versions were tested, not inferred from the manifest alone.
- [ ] Clean-machine solver installation and health check pass.
- [ ] Cloth, multi-object, animated Collider, Rod and Soft Body acceptance scenes pass.
- [ ] Cancel, RAM auto-cancel, invalid cache cleanup and support export pass.
- [ ] Beta update is visible in Beta and Dev channels, never Stable.
- [ ] Support contact and response workflow are ready.
